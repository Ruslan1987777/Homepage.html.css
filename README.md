# Project: Homepage Website

This repository contains a multi-page homepage website built using HTML and CSS. It serves as a simple personal or informational website with structured navigation across several sections.

Pages included

The website consists of 6 main pages:

Index (Home)
Biography
Albums
Video
Downloads
Contacts
Project structure

The project includes the following folders and files:

HTML files – for each of the website pages listed above
CSS file – styles.css for styling and layout
specifications.txt – project requirements and structure notes

Media resources

img/ – contains images used across the website
pdf/ – contains downloadable PDF materials
